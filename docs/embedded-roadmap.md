# Embedded Roadmap For This Repo

This repo already has the right raw material for a stronger embedded-systems portfolio. The next step is to add industry-style structure around the image-processing logic rather than only adding more algorithms.


## Good Fits 

- `lab05` becomes a good home for native kernels and fixed-point or integer experiments.
- `lab06` is a good place to compare Harris implementations and profile windowing strategies.
- `lab07` to `lab09` are strong candidates for pipeline architecture, metrics, frame queues, and failover behavior.
- `embedded_utils/metrics.py` already points in the right direction for embedded profiling discipline.
